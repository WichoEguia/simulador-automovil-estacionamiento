# Simulador Auto

Aplicacion que se usara para gestionar los automoviles que
llegan y salen del estacionamiento.

## Instalación

Reconstruir modulos de node
```
npm install
```

Levantar servidor
```
ng server -o
```

La aplicación se levantará en **localhost:4200**.

## Aplicacion de escritorio
```
npm run electron
```
